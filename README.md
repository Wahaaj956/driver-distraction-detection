# driver-distraction-detection

## Problem
Driver distraction is one of the leading causes of road accidents. Detecting unsafe driver behavior can significantly improve road safety and prevent accidents.

## Approach
Developed a deep learning based image classification model to identify different driver behaviors using image data. The model processes visual input and classifies driver states.

## Technologies Used
- Python
- Tensorflow / Keras
- OpenCV
- Numpy, Pandas

## Result
- Successfully trained a model to classify driver behavior
- Achieved prediction capability on test data
- The model can distinguish between different driver states

## Future Improvements
- Real-time detection using webcam/video feed
- Integration with alert systems for driver safety
- Development as a web application
